# Prompt Injection — Introduction & Exercises

This folder contains exercises and demonstrations about **prompt injection** in the context of cybersecurity and AI.

## What is Prompt Injection?
Prompt injection is an attack technique that manipulates the behavior of a language model or automated assistant by injecting malicious instructions into user input.

## Folder Objectives
- Understand the risks of prompt injection
- Experiment with different attack forms
- Propose mitigation strategies

## Structure
- Example scripts
- Practical cases
- Documentation and resources
