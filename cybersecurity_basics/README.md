# Cybersecurity Basics

This directory introduces the fundamental concepts of cybersecurity. It includes introductory modules, digital forensics, and cryptography basics.

## Structure
- **0x00_introduction_cybersecurity/**: Introduction to cybersecurity concepts and principles
- **0x02_forensic_methodologies/**: Digital forensics methodologies and investigation steps
- **0x03_cryptography_basics/**: Cryptography fundamentals, hashing, and password security

Each subfolder contains scripts, exercises, and documentation to help you understand and practice essential cybersecurity skills.
