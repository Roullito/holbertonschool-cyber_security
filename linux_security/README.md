# Linux Security

This directory focuses on Linux system security, covering permissions, auditing, protocols, and buffer overflow vulnerabilities.

## Structure
- **0x00_linux_security_basics/**: Linux security basics and best practices
- **0x01_permissions_sguid_sgid/**: Permissions, SUID, SGID, and file security
- **0x02_mandatory_access_control/**: Mandatory access control (SELinux, AppArmor)
- **0x03_protocols_servers/**: Securing protocols and servers
- **0x04_buffer_overflow/**: Buffer overflow concepts and exploitation

Each subfolder contains scripts, exercises, and documentation to help you secure and analyze Linux systems.
