# Network Security

This directory covers the fundamentals of network security, including reconnaissance, scanning, traffic analysis, and advanced network attack techniques.

## Structure
- **0x01_passive_reconnaissance/**: Passive information gathering
- **0x02_active_reconnaissance/**: Active scanning and enumeration
- **0x04_nmap_live_hosts_discovery/**: Host discovery with nmap
- **0x05_wireshark_basics/**: Network traffic analysis with Wireshark
- **0x06_nmap_advanced_port_scans/**: Advanced nmap scanning techniques
- **0x07_nmap_post_port_scan_scripting/**: Post-scan scripting and automation

Each subfolder contains scripts, exercises, and documentation to help you understand and practice network security skills.
