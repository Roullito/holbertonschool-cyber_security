# OWASP Top 10

This module covers the most critical web application security risks as identified by the OWASP Top 10 project.

## Topics Covered
- Injection (SQL, NoSQL, OS, etc.)
- Broken Authentication
- Sensitive Data Exposure
- XML External Entities (XXE)
- Broken Access Control
- Security Misconfiguration
- Cross-Site Scripting (XSS)
- Insecure Deserialization
- Using Components with Known Vulnerabilities
- Insufficient Logging & Monitoring

## Learning Objectives
- Understand each OWASP Top 10 risk
- Identify vulnerabilities in web applications
- Apply best practices to mitigate these risks
