# Content Discovery

This module covers techniques for discovering hidden or sensitive content and endpoints in web applications.

## Topics Covered
- Directory and file brute-forcing
- Fuzzing for hidden resources
- Common wordlists and tools
- Risks of exposed sensitive files

## Learning Objectives
- Use tools to discover hidden content
- Understand the risks of exposed files and endpoints
- Apply best practices to secure web content
