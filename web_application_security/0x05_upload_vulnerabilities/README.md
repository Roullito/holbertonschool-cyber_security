# File Upload Vulnerabilities

This module explores vulnerabilities related to file uploads in web applications and how attackers exploit them.

## Topics Covered
- Unrestricted file upload
- Bypassing file type and extension checks
- Remote code execution via uploads
- Mitigation strategies

## Learning Objectives
- Identify file upload vulnerabilities
- Exploit and demonstrate risks
- Implement secure file upload controls
