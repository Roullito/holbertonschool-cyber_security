# Insecure Direct Object Reference (IDOR)

This module covers IDOR vulnerabilities, where attackers gain unauthorized access to resources by manipulating references.

## Topics Covered
- What is IDOR?
- Real-world examples
- Exploitation techniques
- Prevention and secure design

## Learning Objectives
- Identify and exploit IDOR vulnerabilities
- Understand the impact of insecure references
- Apply access control best practices
