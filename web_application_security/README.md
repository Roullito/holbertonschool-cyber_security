# Web Application Security

This directory covers the fundamentals of web application security, including common vulnerabilities, exploitation techniques, and incident response.

## Structure
- **0x00_web_fundamentals/**: Web security fundamentals
- **0x01_owasp_top_10/**: OWASP Top 10 vulnerabilities
- **0x02_burpsuite_fundamentals/**: Burp Suite basics
- **0x03_sql_nosql_injection/**: SQL and NoSQL injection
- **0x04_content_discovery/**: Content discovery techniques
- **0x05_upload_vulnerabilities/**: File upload vulnerabilities
- **0x06_idor/**: Insecure Direct Object Reference (IDOR)
- **0x07_file_inclusion/**: File inclusion vulnerabilities
- **0x09_command_injection/**: Command injection
- **0x0b_web_application_fast_incident_response/**: Fast incident response
- **0x0c_web_application_foresics/**: Web application forensics

Each subfolder contains scripts, exercises, and documentation to help you understand and practice web application security skills.
